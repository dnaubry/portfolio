WebDev Resources

A website containing links to useful resources for web developers.

I wanted to organize the collection of links to websites and books that I found most helpful in improving my web development skills. The ideal solution would make it easy for me to find what I was looking for when I wanted to refer back to one of the resources and would also let me share the collection. Using a CMS would allow me to easily manage the collection of links and would give me experience working with a CMS. I chose to build the website using WordPress, mostly because of its popularity.

I used Scotch Box (a preconfigured Vagrant Box) to develop the WordPress theme locally before uploading it to the live site. The "Blocks" theme displays each link in a block that resizes responsively thanks to CSS3's flexbox. I added search and sort functions to make links easier to find.

Skills and Tools:WordPress, jQuery, HTML, CSS / Sass, PHP, Gulp, Vagrant

Quizzed on TV

A JavaScript quiz web application.

While trying to figure out what to build for my first JavaScript web application, I came across a suggestion that a quiz web app would be a good first project.

The basic objective was to create a quiz that would show one question at a time with multiple answer choices and, after all the questions were answered, the score would be displayed.

Once the basic quiz was functioning, I added additional features including validation, to make sure an answer is selected before the next question is displayed, a back button to allow returning to previous questions, while retaining the selected answers, and expanding the app to support multiple quizzes.

Skills and Tools:JavaScript, HTML, CSS / PostCSS, Handlebars.js, Gulp, webpack
